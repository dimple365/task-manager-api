# Task Manager API

A simple RESTful API built using FastAPI for managing tasks with CRUD operations.

## Features
- Create tasks
- View all tasks
- Update tasks
- Delete tasks

## Tech Stack
- Python
- FastAPI
- SQLite
- SQLAlchemy

## How to Run

1. Install dependencies
2. Run the server
3. Open in browser

## 📌 API Endpoints

- POST /tasks → Create task  
- GET /tasks → Get all tasks  
- PUT /tasks/{id} → Update task  
- DELETE /tasks/{id} → Delete task  

## Description
This project demonstrates basic backend development concepts including REST API design, CRUD operations, and database integration using FastAPI.
